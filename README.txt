# TFG: Aplicación Web de Mobiliario de Oficina

**Autor:** Esteban Campos  
**Ciclo:** Grado Superior de Desarrollo de Aplicaciones Multiplataforma  
**Curso:** 2024-2025

## Descripción

Aplicación web desarrollada en Flask para la venta online de mobiliario de oficina, con gestión de catálogo, usuarios, carrito y pedidos.

## Instalación

1. Clona el repositorio:
   git clone https://github.com/StewieXII/IDEA
2. Instala dependencias:
   pip install -r requirements.txt
3. Configura la base de datos MySQL y carga los scripts.
4. Ejecuta:
   python run.py

## Uso

- Regístrate, inicia sesión y compra productos.
